The CSS selectors that you enter in this field will stop the hyphenatation. They get the class="donthyphenate". Thus the content of the marked HTML element and it's child elements will not be hyphenated. The markup is the same like in a CSS file. You can enter HTML tags or classes (with a leading dot) or ids (with a leading #) or combinations. See the description of setting Hyphenate-Selectors for details.

Be aware: You can't cancel donthyphenate by a hyphenate class in a child element if Hyphenopoly/Hyphenator scripts are controlling the hyphenation. That means that the element P will not be hyphenated in the following code snippet because the surrounding div has a class=donthyphenate.

<div class="someDiv donthyphenate">
	<p class="someP hyphenate">
		Some text that will not be hypheanated!
	</p>
</div>

If you leave this field empty and field Hyphenate-Selectors is empty, too, the plugin will NOT load the Hyphenopoly/Hyphenator scripts.
